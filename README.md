## ⚡ Quick Start

### 1. Install Docker

Make sure Docker and Docker Compose are installed.

---

### 2. Clone this repo

```bash
git clone https://github.com/dutchmen1980/earning-farm.git
cd earning-farm
```

### 3. Add your tokens

Create a `.env` file and add your credentials:
```
EARNAPP_EMAIL=your@email.com
EARNAPP_PASSWORD=yourpassword

EARNFM_TOKEN=your_token_here

HONEYGAIN_EMAIL=your@email.com
HONEYGAIN_PASSWORD=yourpassword

PAWNS_EMAIL=your@email.com
PAWNS_PASSWORD=yourpassword

PACKETSTREAM_CID=your_cid

REPOCKET_EMAIL=your@email.com
REPOCKET_PASSWORD=yourpassword

TRAFFMONETIZER_TOKEN=your_token

PROXYLITE_TOKEN=your_token

PASSIVEAPP_TOKEN=your_token
```

---

### 4. Start everything

```
docker compose up -d
```

If that doesn’t work, try:
```
docker-compose up -d
```

---

Done.
